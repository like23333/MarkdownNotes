# 数据的逻辑结构
```mermaid
graph TD

    A[数据的逻辑结构] --> B[线性结构]
    A --> C[非线性结构]
    
    B --> B1[一般线性表]
    B --> B2[受限线性表]
    B --> B3[线性表推广]
    
    B2 --> B2a[栈和队列]
    B2 --> B2b[串]
    
    B3 --> B3a[数组]
    
    C --> C1[集合]
    C --> C2[树形结构]
    C --> C3[图形结构 / 网状结构]
    
    C2 --> C2a[一般树]
    C2 --> C2b[二叉树]
    
    C3 --> C3a[有向图]
    C3 --> C3b[无向图]
```
1) **集合** 除同属一个集合外没有任何联系
2) **线性结构** 只有一对一
3) **树形结构** 只有一对多
4) **网状或图状结构** 存在多对多关系
