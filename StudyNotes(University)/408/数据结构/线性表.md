# 顺序表
## 顺序表结构
### 静态分配顺序表
```cpp
# define MaxSize 50
template<typename ElemType>
struct SqList{
	ElemType data[MaxSize];
	int Length;
};
```
### 动态分配顺序表
```cpp
#define InitSize 100
template<typename ElemType>
struct SeqList{
	ElemType *data;
	int MaxSize,length;
};
```
## 顺序表初始化
### 静态分配顺序表初始化
```cpp
void InitList(SqList &L){
	L.Length=0;
}
```
### 动态分配顺序表初始化
```cpp
void InitList(SeqList &L){
	L.data=new ElemType[InitSize];
	L.length=0;
	L.MaxSize=InitSize;
}
```
# 单链表
## 单链表结构
```cpp
// 单链表节点定义
struct LNode{
	ElemType data;
	LNode *next;
};
using LinkList = LNode*;
```
## 单链表初始化
### 带头节点的单链表初始化
```cpp
bool InitList(LinkList &L){
	L=new LNode;
	L->next=NULL;
	return true;
}
```
### 不带头结点的单链表初始化
```cpp
bool InitList(LinkList &L){
	L=NULL;
	return true;
}
```
